# 📱 인스파이어 앱 컨셉별 프로토타입 & 레퍼런스 가이드

1. [Joyful] : The Digital Playground
컨셉 목표: 리조트의 '오로라'와 '아레나'가 주는 시청각적 압도감을 앱으로 전이시켜 사용자의 기대감을 극대화함.

배경: 인스파이어는 단순한 숙박 시설이 아닌 **'엔터테인먼트 리조트'**입니다. 15,000석 규모의 아레나, 세계 최대 규모의 디지털 아트 거리 '오로라', 사계절 워터파크 '스플래시 베이' 등 이곳을 찾는 고객의 최우선 목적은 **'즐거움(Entertainment)'**에 있습니다.

도출 근거: "리조트에서 느끼는 압도적인 시각적 즐거움이 앱에서도 끊김 없이 이어져야 한다"는 니즈에서 출발했습니다. 앱이 단순히 예약 도구가 아니라, 축제의 시작점이 되어야 한다는 논리입니다.

🎨 비주얼 전략 (Visual Strategy)
Tone & Manner: 다크 모드(Dark Mode), 고대비(High Contrast), 시네마틱 루핑 비디오.

Graphic: 오로라의 핑크·퍼플·블루 그라데이션을 포인트 컬러로 사용.

이미지 컨셉: 실사 위주의 고화질 영상과 역동적인 인물 사진(환호하는 관객 등), 직접 공간을 둘러보는 듯한 3D 렌더링 이미지.

👆 모바일 특화 인터랙션
자이로 센서 틸트 (Gyro Tilt): 기기를 좌우로 기울이면 배경의 오로라 영상 시점이 미세하게 변하는 3D 뎁스 효과.

롱프레스 프리뷰 (Peek & Pop): 공연 썸네일을 꾹 누르면 배경음과 함께 짧은 하이라이트 영상이 팝업으로 재생.

디지털 파티클 햅틱 (Haptic Feedback): 예약 완료 버튼 클릭 시 화면에 디지털 폭죽이 터지며 경쾌한 진동 피드백 제공.

### 🎬 시연 시나리오 (축제의 시작)
1.  **진입:** 앱 실행 시 실제 오로라 쇼 영상이 풀 화면으로 재생되며 현장감 부여.
2.  **인지:** 메인 화면에 공연 입장 카운트다운 위젯 노출.
3.  **액션:** 티켓 활성화 시 화려한 인터랙션과 함께 모바일 패스 등장.

🔗 레퍼런스 URL
비주얼 무드: https://www.wearekaleida.com/ (화려한 영상 활용 사례)

브랜드 시스템: https://www.breachbunny.com/ (강렬한 다크 UI와 콘텐츠 스택)

인터랙션 참고: https://brand.socar.kr/ (역동적인 트랜지션 사례)

2. [Refined] : The Signature Collection
컨셉 정의: **'프리미엄의 정수'**와 **'전문적인 큐레이션'**을 시각화. 세련된 도시적 감성과 고품격 서비스를 강조.

배경: 최근 대주주가 모히건(Mohegan)에서 **베인캐피탈(Bain Capital)**로 변경되었습니다. 사모펀드 체제에서는 글로벌 스탠다드에 부합하는 전문성, 효율적인 프리미엄 서비스, 자산 가치의 극대화가 중요해집니다.

도출 근거: 세련되고(Sophisticated) 정제된(Refined) 하이엔드 이미지를 구축하여 글로벌 부티크 호텔 수준의 브랜드 가치를 제안하기 위함입니다. 이는 수익성을 중시하는 새로운 경영진의 지향점과 맞닿아 있습니다.

🎨 비주얼 전략 (Visual Strategy)
Tone & Manner: 하이엔드 부티크(High-end Boutique), 모던 럭셔리, 미니멀리즘.

Key Color: Midnight Blue, Champagne Gold, Deep Charcoal.

Graphic Assets: 폰트 중심의 타이포그래피(Serif), 전문 큐레이터가 찍은 듯한 감각적인 스틸컷.

👆 모바일 특화 인터랙션
심리스 시트 (Seamless Sheet): 모든 정보가 하단에서 우아하게 올라오는 슬라이딩 시트 구조로, 화면 전환 피로도 최소화.

스마트 필터링 (Smart Sorting): 사용자의 과거 이용 패턴을 바탕으로 '당신만을 위한 시그니처 경험'을 상단에 고정 노출.

매끄러운 트랜지션: 화면 이동 시 페이드 아웃(Fade-out) 대신 콘텐츠가 스르륵 겹쳐지는 '오버레이 애니메이션'으로 고급스러운 느낌 강조.

### 🎬 시연 시나리오 (품격 있는 휴식)
1.  **환대:** 사용자의 이름을 부르는 정중한 웰컴 메시지와 함께 여백의 미가 강조된 메인 노출.
2.  **탐색:** 전문 큐레이터가 엄선한 시그니처 다이닝 리스트를 에디토리얼 스타일로 감상.
3.  **요청:** 컨시어지 탭에서 와인잔 세팅 등을 슬라이드 제스처로 우아하게 요청.

🔗 레퍼런스 URL
비주얼 무드: https://www.blesscollectionhotels.com/en/ (극도의 미니멀리즘과 럭셔리 무드)

브랜드 시스템: https://badruttspalace.com/en/winter/ (고급 컨시어지 서비스의 정제된 UI)

인터랙션 참고: https://culinaryodissey.uprock.pro/ (페이드아웃 + 오버레이 애니메이션 사례)

3. [Customized AI] : The Smart Navigator
컨셉 목표: 복잡한 대형 리조트 인프라를 AI가 개인별 맞춤 동선으로 정리. '내 손안의 완벽한 비서'로서의 효율성 강조.

배경: 인스파이어는 부지가 매우 넓고 시설이 복잡합니다. 수많은 레스토랑, 공연, 카지노, 쇼핑몰 사이에서 사용자는 **'선택의 과부하'**와 **'이동의 피로감'**을 느낍니다.

도출 근거: 거대한 리조트 안에서 길을 헤매지 않고, 수천 개의 옵션 중 나에게 맞는 것만 골라주는 **'지능형 가이드'**가 필수적입니다. 최신 기술인 AI를 통해 이 복잡성을 해결하고 **'초개인화된 편의성(Seamless Convenience)'**을 제공하겠다는 기술적 해법입니다.

🎨 비주얼 전략 (Visual Strategy)
Tone & Manner: 클린 미니멀리즘, 글래스모피즘(Glassmorphism), 미래지향적 무드.

Graphic: 3D 아이소메트릭(Isometric) 지도, 정교한 데이터 위젯, 투명도 레이어.

이미지 컨셉: 심플한 3D 그래픽 에셋과 정보 중심의 인포그래픽.

👆 모바일 특화 인터랙션
다이내믹 모핑 (Dynamic Morphing): 투숙 상태(체크인/아웃)에 따라 하단 탭 바와 메인 카드가 실시간으로 형태 변환.

핀치 투 맵 (Pinch to Map): 일정 카드를 두 손가락으로 벌리면 즉시 실시간 이동 동선이 포함된 3D 지도로 전환.

지능형 햅틱 피드백 (Precise Haptic): AI 제안 수락 시 기계적이고 명확한 진동을 주어 기술적 신뢰도 전달.

### 🎬 시연 시나리오 (똑똑한 동행)
1.  **인식:** 로비 진입 시 AI가 위치를 인식해 '스마트 체크인' 위젯을 최상단에 자동 노출.
2.  **제안:** 현재 대기 시간이 가장 짧은 시설(예: 스플래시 베이)을 AI가 실시간 제안.
3.  **가이드:** 제안 카드를 핀치 아웃하여 3D 지도로 전환, 목적지까지 최적 동선 안내.

🔗 레퍼런스 URL
비주얼 무드: https://linear.app/ (기술적 정교함과 세련미를 동시에 갖춘 UI의 정석)

브랜드 시스템: https://gemini.google.com/app?hl=ko (데이터 기반 위젯의 위계 정립, AI 채팅을 내세운 미니멀리즘 UI)

인터랙션 참고: https://studiolumio.com/ (지능형 서비스의 실시간 피드백 모션)

# 📱 인스파이어 앱 시안 제작 세부 스택 가이드

인스파이어 엔터테인먼트 리조트 앱 구축 제안을 위한 모바일 디자인 사양서입니다. **iPhone 15/16 Pro(393x852px)** 해상도에 최적화하여 시연 시 최고의 퀄리티(Pixel Perfect)를 보장하도록 설계되었습니다.

---

## 1. 기본 프레임 및 해상도 (Frame & Resolution)
프리미엄 리조트 제안에 적합한 하이엔드 기기 사양을 기준으로 작업합니다.

| 항목 | 상세 스택 정보 | 비고 |
| :--- | :--- | :--- |
| **기준 기기** | **iPhone 15 / 16 Pro** | 프리미엄 이미지 구축 |
| **뷰포트 사이즈** | **393 x 852 px** | Figma 프레임 기준 |
| **렌더링 해상도** | **1179 x 2556 px (@3x)** | 실제 기기 출력 해상도 |
| **화면 비율** | **19.5 : 9** | 세로형 인피니티 디스플레이 |
| **인터랙션 도구** | ProtoPie (센서 활용), Figma (Smart Animate) | |

---

## 2. 레이아웃 그리드 시스템 (Layout Grid)
일관된 시각적 흐름을 위해 8단위(8-pt) 그리드 시스템을 적용합니다.

* **Type:** 4-Column Grid
* **Margin (좌우 여백):** * **Joyful / AI 컨셉:** 20px (몰입감과 정보량 확보)
    * **Refined 컨셉:** 24px (고급스러운 여백의 미 강조)
* **Gutter (단 간격):** 16px
* **Vertical Rhythm:** 모든 간격(Spacing)은 4px / 8px의 배수를 기본으로 사용

---

## 3. 시스템 UI 영역 가이드 (System UI Area)
실제 구동되는 앱의 생동감을 위해 반드시 준수해야 하는 안전 영역입니다.

* **상단 상태바 (Status Bar):** **54 px** (노치 영역 포함)
* **하단 홈 인디케이터 (Home Indicator):** **34 px** (제스처 영역)
* **권장 하단 탭바 높이 (Tab Bar):** **83~90 px** (홈 인디케이터 포함)
* **안전 영역 (Safe Area):** 핵심 인터랙션 요소는 상단 54px 및 하단 34px 영역을 침범하지 않도록 배치

---

## 4. 컨셉별 레이아웃 디테일 스택

### [Concept 1: Joyful]
* **Layout Strategy:** **Edge-to-Edge** (전면 풀 배너 활용)
* **Component Style:** 배경 영상의 가시성을 위해 투명도(Opacity)가 적용된 오버레이 위젯과 글래스모피즘 적용.

### [Concept 2: Refined]
* **Layout Strategy:** **Editorial Grid** (매거진 스타일 레이아웃)
* **Component Style:** 세리프 폰트 중심의 타이포그래피 활용, 최소한의 선(1px)과 은은한 드롭 쉐도우(Blur: 10~20) 사용.

### [Concept 3: Customized AI]
* **Layout Strategy:** **Modular Dashboard** (벤토 그리드 시스템)
* **Component Style:** 3D 그래픽 에셋 배치, 하단 탭 바 상단에 AI 전용 플로팅 액션 버튼(FAB) 배치.

---

## 5. 시연 및 출력 설정 (Prototyping & Export)

* **Prototyping:**
    * 프레임 높이는 콘텐츠에 따라 늘리되, **Scrolling 시작점은 852px**로 고정.
    * GNB와 Tab Bar는 스크롤 시 상/하단에 고정(`Fix position when scrolling`).
* **Exporting:**
    * 제안서 문서 삽입용: **PNG @2x 또는 @3x** (고해상도 유지)
    * 시연 영상용: 직접 화면을 퍼블리싱해 URL 시연 녹화
